# Exploratory Data Analysis (EDA) Project

## Overview
This project demonstrates a complete Exploratory Data Analysis (EDA) workflow using Python. It focuses on loading a dataset, inspecting its structure, checking for missing values and duplicates, analyzing relationships, identifying trends and outliers, and visualizing data effectively.  

The project uses a sample dataset (`tips`) from the `seaborn` library, making it easy to run without requiring an external CSV file.

---

## Features
- Dataset inspection (info, statistical summary)
- Check for missing values and duplicate rows
- Analyze number of unique values per column
- Compute correlation matrix for numeric features
- Identify trends and outliers using boxplots
- Visualize relationships between numeric features using pairplots
- Visualize categorical columns using count plots

---

## Technologies & Libraries

- pandas
- matplotlib
